# springboot-security-jwt-mybatis

## 개요

Spring Boot Security 6 JWT Mybatis 
- access token / refres token 
- JPA 안쓰고 Mybatis 사용 

## 개발환경

- Spring Tool Suite 4 (4.22.0)
- SpringBoot 3.2.5
- SpringSecurity 6
- jsonwebtoken 0.12.3
- JDK17
- Mybatis
- Mysql
- Maven
 
## 설명

Spring Boot Security 6 JWT Mybatis 샘플 간단한 테스트 화면 추가
